# 簡易表記チェックツール（ベータ）

## 使い方
- `index.html` をブラウザで開いて入力し、「チェックする」を押します。
- 右側（下部）の「推奨表記に直した文章（プレビュー）」に自動修正案が表示されます。

## 重要：file:// 直開きは不可
このツールは `fetch()` で `rules.json` を読み込むため、`file://` で直接開くとブラウザのCORS制約で動きません。
必ず HTTP/HTTPS で配信してください（例：VS Code Live Server / Python http.server / GitHub Pages / レンタルサーバー）。
